O que cada Usuario pode fazer?

Uma pequena análise do que eu acredito que será viável a aplicação a respeito das visões e funções do projeto.

Diretor Geral

  • Permissões

  • Visualizar os dados de todas as tabelas.
  • Inserir, remover, atualizar dados de qualquer tabelas

  • Possíveis visões

  • Quantidade arrecadada por especialidades cadastrada na clínica.
  • Quantidade de funcionários por cargos na clínica.
  • Quem são os funcionários supervisores.

  • Possíveis functions

  • Cadastrar um novo paciente.
  • Remover um paciente.
  • Atualizar o número de telefone de um paciente.
  • Aumentar o salário de uma categoria por tipo.
  • Remover a função de Supervisor de um funcionário.
  • Adicionar a função de Supervisor de um funcionário.
  • Consultar os médicos de uma determinada especialidade.

Atendente

  • Permissões

  • Visualizar os dados na tabela Paciente e referente ao seu telefone.
  • Inserir utilizando a função para tal , atualizar dados da tabela Paciente.
  • Visualizar os dados na tabela de Especialidade.

  • Possíveis visões

  • Qual o Nome e o número de telefone de um determinado paciente.
  • Catálogo das especialidades e o preço da consulta.
  • A quantidade de pacientes de cada endereço (Bairro/Cidade).

  • Possíveis functions

  • Cadastrar um novo paciente.
  • Remover um paciente.
  • Atualizar o número de telefone de um paciente.
  • Consultar os médicos de uma determinada especialidade.

Médico

  • Permissões

  • Inserir, dados na tabela Receita utilizando a função para tal.
  • Visualizar os dados na tabela de Especialidade.

  • Possíveis visões

  • Nenhuma em mente

  • Possíveis functions

  • Cadastrar uma nova receita.
  • Obter todas as receitas que este receitou.
  • Obter a quantidade de pacientes consultados pelo tal.
  • Obter o número de telefone de um determinado paciente
